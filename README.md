# 🌿 MenuGreen Privacy Policy

Privacy Policy cho ứng dụng **MenuGreen** - Mobile app đặt đồ ăn healthy với tính năng tính calories và gợi ý món ăn.

## 📋 Thông tin

- **App Name:** MenuGreen
- **Package ID:** `com.menugreen.app`
- **Nền tảng:** Android (Flutter)
- **Backend:** Firebase + AWS RDS PostgreSQL
- **Last Updated:** July 11, 2026

## 🚀 Công nghệ sử dụng trong app

- ✅ Email/Password authentication (Firebase Auth)
- ✅ Push notifications (Firebase Cloud Messaging)
- ✅ Database: AWS RDS PostgreSQL
- ✅ Thanh toán: SePay
- ✅ Tính năng tính calories và gợi ý món ăn
